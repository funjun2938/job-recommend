# Ultragoal: 수익화 업그레이드 — "진짜 돈벌만한 수준"

## 목표
이직추천 서비스를 실제 수익을 창출할 수 있는 SaaS 제품으로 전환한다.

## 수익 모델
| 티어 | 가격 | 기능 |
|------|------|------|
| Free | 무료 | 분석 1회, 기본 결과 |
| Pro | 9,900원/월 | 무제한 분석, 자소서 5회/월, 면접 질문, 결과 저장 30일 |
| Team | 49,000원/월 | Pro + B2B HR 리포트, API 접근 |

## Stories (순서대로 실행)
- G001: Stripe 결제 + Pricing 페이지
- G002: Supabase Auth (Google + Kakao 소셜 로그인)
- G003: 프리미엄 기능 게이팅 (사용량 제한 + 업그레이드 유도)
- G004: 사용자 대시보드 (분석 히스토리)
- G005: Resend 이메일 알림 실제 발송
- G006: SEO 최적화 (직군별 랜딩 + sitemap + structured data)

## 완료 조건
- Stripe checkout 실제 작동
- 프리미엄 기능 게이팅 작동
- 사용자 로그인 흐름 완성
- 이메일 실제 발송
- SEO 메타태그 + sitemap 완성
